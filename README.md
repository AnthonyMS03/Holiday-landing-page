# Holiday-landing-page
Simple Holiday landing page
-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
This project is a simple holiday-themed landing page showcasing hero imagery, animations, content management, and some interactive components.
It demonstrates front-end development, UX design, content organization, and image optimization.

  - Full-screen holiday hero image

  - Soft gold headline using Playfair Display

  - Dark overlay for a luxury aesthetic

  - Smooth fade-in animation

  - Call-to-action button with hover transitions
-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
Features

Snowfall Animation

  - Subtle animated snowflakes using lightweight JavaScript

Live Countdown Timer

  - Auto-updating timer to December 25th

  - Shows days, hours, minutes, and seconds

Featured Product Gallery

Responsive grid layout

Smooth hover lift animation
-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
How to Run This Project:

1. clone the repository

2. Right-click index.html and select: “Open with Live Server” (recommended)
